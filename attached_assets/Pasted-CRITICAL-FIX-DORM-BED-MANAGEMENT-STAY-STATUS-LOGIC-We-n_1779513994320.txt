====================================================
1. STAY STATUS ENGINE
====================================================

Each booking must belong to ONLY ONE state.

----------------------------------------------------
ARRIVING TODAY
----------------------------------------------------

Condition:
checkInDate === today

Example:
Guest arrives today but not checked in yet.

Status label:
"Arriving Today"

Color:
Yellow

----------------------------------------------------
IN HOUSE / CONTINUING STAY
----------------------------------------------------

Condition:
today >= checkInDate
AND
today < checkOutDate
AND
checkedIn = true

Status label:
"In House"

Color:
Green

----------------------------------------------------
CHECKOUT TODAY
----------------------------------------------------

Condition:
checkOutDate === today

Status label:
"Checkout Today"

Color:
Red

This guest still occupies bed UNTIL checkout completes.

----------------------------------------------------
FUTURE RESERVATION
----------------------------------------------------

Condition:
today < checkInDate

Status label:
"Upcoming"

Color:
Blue

----------------------------------------------------
COMPLETED
----------------------------------------------------

Condition:
today > checkOutDate
OR
checkedOut = true

Do not show in active dorm screen.
