PHASE 10 – 🤖 AI Inventory Investigator

Objective:

Build an AI-powered troubleshooting assistant that automatically investigates inventory, OTA sync, availability mismatches, booking discrepancies, and calendar inconsistencies.

Location:

Channel Manager
→ Verify Property
→ AI Investigator

or

Add button:

🤖 Investigate

inside:

* OTA Test
* Inventory Consistency
* Reconciliation

---

Problem

Today staff must manually inspect:

* OTA Test
* Inventory Consistency
* Reconciliation
* Sync Logs
* Bookings
* Room Mapping

to understand why inventory is incorrect.

The AI Investigator should perform this analysis automatically.

---

User Experience

User clicks:

🤖 Investigate

or asks:

"Why is Deluxe Room showing 3 available on 10 June?"

---

AI Investigation Engine

The AI should automatically collect:

1. OTA Health Audit
2. Inventory Consistency Results
3. Inventory Reconciliation Results
4. Recent Sync Logs
5. Failed Push Logs
6. Room Mapping
7. Rate Plan Mapping
8. OTA Test Results
9. Booking Breakdown
10. Blocked Room Data
11. Recent OTA Reservations
12. Recent Offline Reservations

Expected Responses

Case 1

Inventory Mismatch

Output:

🔴 Investigation Complete

Room:
Deluxe Room

Date:
10 June

Hostezee:
4 Available

AioSell:
5 Available

Root Cause:
Walk-in booking reduced inventory in Hostezee.

Inventory push failed.

Last Error:
HTTP 429 Too Many Requests

Impact:
OTA inventory is stale.

Recommended Action:
Run inventory sync and verify AioSell update.

---

Case 2

Room Mapping Issue

Output:

🔴 Investigation Complete

Root Cause:
INVALID_ROOM_CODE detected.

Room:
4-bed Female Dormitory

AioSell does not recognise the configured room code.

Recommended Action:
Correct room mapping in Channel Manager.

---

Case 3

Rate Plan Issue

Output:

🟡 Investigation Complete

Root Cause:
INVALID_RATEPLAN_CODE detected.

Recommended Action:
Verify rate plan code in AioSell and Hostezee.

Case 4

Everything Healthy

Output:

🟢 Investigation Complete

No issues detected.

Inventory:
Synchronized

OTA Connectivity:
Healthy

Recent Inventory Push:
Success

Last OTA Reservation:
Imported Successfully

No action required.
