We need to improve the Active Booking section in the PMS to allow staff to quickly add extra services to a guest’s booking without leaving the booking screen.

Currently, if staff wants to add a service like taxi, bonfire, laundry, or extra food, they must go to the Add-on Services section and manually link it to the booking. This is slow and not convenient during operations.

New Feature Requirement:

Add an “Add Service” or “Extra Service” button inside each booking card in the Active Bookings page.

Location:
On the right side of each booking row/card where options like View Details or ID verification currently exist.

Workflow:

1. Staff opens Active Bookings.

2. Each booking row should have a new button:
   "Add Service"

3. When clicked, a modal window should open showing available service types such as:

   * Taxi
   * Bonfire
   * Laundry
   * Extra Bed
   * Food Order
   * Custom Service

4. Staff selects a service and fills:

   * Service Type
   * Description (optional)
   * Price / Fare
   * Date & Time
   * Notes (optional)

5. When saved:

   * The service is attached to that specific booking.
   * The service appears under the booking details.
   * The service is automatically added to the guest’s bill.

Billing Integration:

When the guest checks out:
All added services should automatically appear in the final bill along with the room charges.

Example Bill:

Room Charges – ₹5000
Food Orders – ₹1200
Taxi Service – ₹1500
Bonfire – ₹800

Total – ₹8500

Service Management:

Add a settings section called “Service Types” where admin can create service categories such as:

Taxi
Bonfire
Laundry
Extra Bed
Local Tour
Custom Service

Each service type should allow a default name and optional default price.

Reporting:

All services added to bookings should be tracked so the admin can generate reports like:

* Total Taxi Revenue
* Total Bonfire Revenue
* Total Extra Service Revenue

This feature should help hotel staff quickly add services directly from the Active Booking screen and ensure all services are automatically included in the guest billing and revenue reports.
