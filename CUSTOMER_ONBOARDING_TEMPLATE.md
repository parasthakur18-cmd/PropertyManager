## üìö Key Features & How to Use

### A. Creating a Booking

**Steps:**
1. Go to **Bookings** ‚Üí "New Booking"
2. Select room(s)
3. Select dates (check-in & check-out)
4. Enter guest details:
   - Name
   - Email
   - Phone number
5. Select booking type:
   - Single Room
   - Group Booking (multiple rooms)
   - Dormitory (multiple beds in same room)
6. Add special requests (optional)
7. Save

**Result**: 
- Booking confirmation email is automatically sent to guest
- Guest receives QR code for self check-in
- Booking appears in your dashboard

### B. Guest Self Check-in

**How it works:**
1. Guest receives booking confirmation email with QR code
2. On arrival day, guest scans QR code or visits check-in page
3. Guest verifies email and uploads ID proof
4. System marks guest as checked-in ‚úÖ

**Benefits:**
- No waiting time for guests
- Reduces staff workload
- Automatic ID verification
- Instant status update in system

**Guest Check-in URL:**
```
[Your URL]/guest-self-checkin
```

### C. Managing Active Bookings

**Real-time Dashboard:**
1. Go to **Active Bookings**
2. See all checked-in guests
3. View:
   - Room details
   - Guest information
   - Billing details
   - Orders placed
4. Actions:
   - Merge caf√© bills (for in-house guests)
   - Generate final bill
   - Process checkout

### D. Menu & Orders

**Set up your menu:**
1. Go to **Menu**
2. Create categories (Breakfast, Lunch, Dinner, etc.)
3. Add items with:
   - Name
   - Description
   - Price
   - Image
   - Variants (if applicable)
   - Add-ons

**Process orders:**
1. Go to **Orders**
2. Create room service order for guests
3. Track order status:
   - Pending ‚Üí Preparing ‚Üí Ready ‚Üí Completed
4. Charges automatically added to guest bill

### E. Financial Management

**Track your finances:**
1. Go to **Financials**
2. View:
   - Revenue by property
   - Revenue by booking source
   - P&L statements
   - Expense tracking
3. Generate reports for accounting

**Create bills:**
1. When guest checks out
2. Click "Checkout Guest"
3. System auto-calculates:
   - Room charges
   - Food charges
   - Extra services
   - Taxes (GST/Service Charge)
4. Apply discount if applicable
5. Generate bill PDF
6. Mark as paid/pending

### F. Enquiry Management

**Convert prospects to bookings:**
1. Go to **Enquiries**
2. Create new enquiry:
   - Guest name
   - Contact info
   - Dates
   - Rooms needed
3. Track enquiry status
4. Convert to booking when confirmed

### G. Staff Management

**Add staff users:**
1. Go to **Settings** ‚Üí **Users**
2. Click "Add User"
3. Assign role:
   - Admin: Full system access
   - Manager: Can view reports (read-only)
   - Staff: Can manage bookings and orders
   - Kitchen: Can only see orders
4. Assign properties
5. Staff receives login credentials

## üìû Support & Help

### Common Issues

**Q: Guest didn't receive confirmation email?**
A: Check guest email address is correct. Resend email from booking details.

**Q: How to merge caf√© bills?**
A: During checkout, use "Merge Caf√© Orders" button to add caf√© charges to room bill.

**Q: Can I edit a completed booking?**
A: Only pending/confirmed bookings can be edited. Checked-out bookings are locked for audit.

**Q: How to track pending payments?**
A: Go to Pending Payments tab. Mark as paid when received.

**Q: Can guests check in without the QR code?**
A: Yes! They can manually enter booking ID on check-in page.
